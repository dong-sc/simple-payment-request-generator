# 請款單製作工具 Simple Payment Request Generator

一個給自由工作者與小型工作室使用的簡易請款單製作工具。

不用登入、不需安裝、不需要後端。
打開網頁，填寫請款資訊，即可產生一份乾淨、可列印、可另存 PDF 的請款單。

這個工具的目標不是取代會計系統、發票系統或金流服務，而是把大家原本就會做、但很煩的請款文件製作流程，變成更低門檻的小工具。

## 專案特色

- 純前端工具，不需要後端、資料庫或帳號系統
- 使用填格子的方式輸入請款資料
- 即時產生正式文件風格的請款單預覽
- 支援列印與透過瀏覽器另存 PDF
- 支援複製文字版請款內容
- 使用 localStorage 自動暫存資料
- 支援 TWD / USD / JPY / HKD / CNY
- 支援付款方式與銀行帳戶資訊
- UI 文案使用繁體中文

## 適合誰使用

- 自由工作者
- 小型工作室
- 接案者
- 設計師、攝影師、剪輯師
- 顧問、講師與服務型工作者
- 小型服務業者
- 需要快速製作請款單的人

## 功能列表

- 請款單標題、編號、日期、付款期限與幣別設定
- 對應報價單編號與專案名稱欄位
- 請款方資訊欄位
- 客戶資訊欄位
- 多筆請款項目新增、刪除與編輯
- 自動計算請款項目小計、折扣、稅額、本次請款金額、已收款與尚待支付金額
- 稅率可設定為 0%，並顯示未稅 / 免稅
- 付款方式、銀行資訊、付款備註與匯款後回覆方式
- 請款備註、付款條款與交付說明
- 預設文字範本快速加入
- 列印時只顯示請款單內容
- 清空表單並恢復預設值

## 請款與付款資訊功能說明

本工具支援基本請款資訊、請款項目與付款資訊欄位。

付款方式可選擇銀行轉帳、現金、支票、PayPal 或其他。若選擇其他，可自行輸入付款方式。

銀行資訊包含銀行名稱、分行名稱、銀行代碼、帳號與戶名。也可填寫匯款後回覆方式，例如「匯款後請回覆末五碼，謝謝。」

本工具不串接金流、不處理線上付款，也不開立發票。它只協助你產生一份乾淨、可列印、可傳給客戶的請款文件。

## 隱私說明

本工具不需要登入，也不會將資料上傳到任何伺服器。
所有輸入內容僅儲存在你的瀏覽器 localStorage 中。
若你按下清空表單，資料會從瀏覽器暫存中移除。

## 本機開發方式

```bash
npm install
npm run dev
```

開發伺服器啟動後，依終端機顯示的網址開啟瀏覽器即可。

## 建置

```bash
npm run build
```

## GitHub Pages 部署方式

本專案已包含 GitHub Actions workflow：`.github/workflows/deploy.yml`。

使用方式：

1. 將程式推送到 GitHub repository。
2. 到 repository 的 Settings → Pages。
3. Source 選擇 GitHub Actions。
4. push 到 `main` branch 後會自動執行 build 並部署 `dist`。

部署到 GitHub Pages 時，`vite.config.ts` 使用：

```ts
base: "/simple-payment-request-generator/"
```

若你的 repository 名稱不同，請同步調整 base 路徑。

## Vercel 部署方式

Vercel 可直接匯入此 repository 並使用預設 Vite 設定部署。

若部署到 Vercel 根路徑，`vite.config.ts` 可視情況改為：

```ts
base: "/"
```
