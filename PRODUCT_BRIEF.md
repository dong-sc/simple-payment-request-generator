# Product Brief

## 中文名稱

請款單製作工具

## English Name

Simple Payment Request Generator

## Repo Name

simple-payment-request-generator

## Core Idea

A simple no-login payment request generator for freelancers and small studios.

This tool helps users create a clean payment request document through a familiar fill-in-the-form workflow.

It is not a full accounting system.

It is not an invoice issuing system.

It is not a CRM.

It is not a backend admin panel.

It is a small everyday work tool.

## Target Users

- freelancers
- small studios
- solo service providers
- designers
- photographers
- editors
- consultants
- lecturers
- anyone who needs to create simple payment request documents quickly

## Main Use Case

A user has completed a project or reached a payment milestone.

They open the tool, fill in the payment request information, preview the document, and print, save as PDF, or copy a plain-text payment request message.

The tool should feel familiar and low-pressure.

## Key Features

- payment request information input
- issuer information
- client information
- payment item table
- subtotal calculation
- tax rate
- received deposit
- remaining balance
- payment due date
- bank account information
- payment instructions
- payment request preview
- copy plain text payment message
- print / save as PDF
- localStorage draft persistence

## Success Criteria

A first-time user should be able to create a usable payment request document within 3 minutes.

The tool should not require login, explanation, or setup.

The print output should look clean enough to send to a client.
